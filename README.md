# AdSense Checker Tool

A free online tool to quickly check if a website has **Google AdSense** enabled or not.

Live Demo: **[https://adsense-checker.netstudy.in/](https://adsense-checker.netstudy.in/)**

## Features

- Automatically detects AdSense publisher ID from the website source code
- Checks for official Google AdSense script presence
- Fast and lightweight – no login or signup required
- Works on any public website
- Simple and user-friendly interface
- 100% free to use

## How It Works

1. Enter any website URL (e.g., https://example.com)
2. Click "Check AdSense"
3. The tool scans the page source for common AdSense patterns:
   - `ca-pub-` publisher ID
   - Google AdSense script tags
4. Shows instant result: **Active**, **Not Found**, or **Error**

## Why Use This Tool?

- Publishers often want to know if competitor sites are monetized with AdSense
- Bloggers & SEO professionals use it for quick site audits
- Completely free alternative to manual source code inspection
